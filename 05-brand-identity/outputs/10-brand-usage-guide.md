---

## Colours

| Name | Hex | Use |
|------|-----|-----|
| Forest (primary) | `#0e5c42` | Brand colour. Headers, primary buttons, wordmark, trust badge: Reliable. |
| Amber (accent text) | `#8a5c1a` | Accent. Links on white, trust badge: Established. Passes WCAG AA on white. |
| Amber Light (decorative) | `#c97d2e` | Borders, badge backgrounds, large icons only. Never for normal body text on white. |
| Ink | `#1a1a18` | All body text. |
| Slate | `#484844` | Secondary text. |
| Dust | `#6e6e6a` | Tertiary/metadata text. Trust badge: New. |
| Parchment | `#f7f6f4` | Card backgrounds, alternating rows. |
| Error | `#b91c1c` | Validation, destructive actions only. |

One primary (Forest). One accent (Amber family). No third chromatic colour. No gradients.

---

## Typography

**Headings:** Source Serif 4 · weights 400 / 500 / 600 · Google Fonts (SIL OFL 1.1, free)
**Body & UI:** Source Sans 3 · weights 400 / 500 / 600 · Google Fonts (SIL OFL 1.1, free)

Minimum text size: 12px (xs) for supplementary text only. All instructional and error text ≥ 14px.

---

## Logo rules

- Minimum wordmark width: 120px
- Clear space: one cap-height on all sides
- Use on: white, Parchment, Forest (reversed white version), Amber Subtle
- Never: recolour, stretch, rotate, add effects, use below minimum size, lock up with imagery

---

## Accessibility

All text/background pairings pass WCAG 2.2 AA. Full audit in `09-accessibility-audit.md`.
Focus ring: 3px solid Forest, 3px offset. Never suppress focus styles.
Colour is never the sole differentiator — all colour-coded states carry a text label.

---

## What this brand is not

Not a startup: no rounded-lowercase wordmark, no gradient, no exclamation marks.
Not a heritage cricket brand: no red+cream, no stumps, no crossed bats, no Victorian illustration.
Not a sports-media product: no urgency, no neon, no franchise palettes.
